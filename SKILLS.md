# Project Rules

## Stack

- Next.js App Router
- TypeScript
- Tailwind
- shadcn/ui
- TanStack Query
- MongoDB + Mongoose

## Code Style

- Prefer server actions
- Prefer async/await
- Use Zod for validation
- Avoid any
- Reusable components only

## TanStack Query

- Use optimistic updates
- Query keys in separate files
- Mutations in hooks

## UI

- Use shadcn patterns
- Accessible components
- Mobile first

## Forms

- TanStack Form
- Zod validation

## Naming

- camelCase variables
- PascalCase components
